# Hyperloop Train Booking System

A modern web application for booking hyperloop train tickets with a sleek, glass-morphism design.

## Features

- Modern UI with glass-morphism effects
- Responsive design for all devices
- Dark theme with gradient accents
- Interactive booking interface
- Real-time route visualization

## Tech Stack

- HTML5
- CSS3 (with modern features like backdrop-filter)
- JavaScript
- Responsive Design

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start booking your hyperloop journey!

## Design Elements

- Glass-morphism cards and headers
- Gradient buttons with hover effects
- Dark theme with overlay
- Responsive navigation
- Modern form controls

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
